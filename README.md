# 🕒 US Market Clock 美股时钟小窗

> 一个轻量、常驻角落的美股交易时段悬浮小窗  
> 适用于 Tampermonkey 等  
> 纯净无广，作者自用

---

## ✨ 功能特性

### 🔹 交易时段识别 + 倒计时
* 自动判断当前处于：
  * **盘前 / 盘中 / 盘后 / 休市**
* 显示 **下一时段倒计时**（精确到秒）
* 内置常见 NYSE 假日算法（best-effort）

> 注意：**不含半日市/临时休市/突发休市** 等特殊情况。

### 🔹 时区显示一键切换（NY / 本地）
* 标题旁徽章按钮 **NY/LOC**：
  * NY：以美东时区显示表格时间
  * LOC：以电脑当前时区显示表格时间
* 交易状态与倒计时始终以 **美东（交易所）** 为准，避免时区切换导致计算错误

### 🔹 UI & 交互
* 悬浮在任意网页上，不影响页面本身
* **可拖拽移动**，并自动记忆位置
* **点击标题栏折叠/展开**（无需额外按钮）
* 紧凑设计，适合常驻屏幕角落

---

## 📦 安装方式

### 1️⃣ 安装油猴插件
* Chrome / Edge：Tampermonkey
* Firefox：Tampermonkey / Violentmonkey

### 2️⃣ 新建脚本
* 打开 Tampermonkey → 新建脚本
* 粘贴本项目提供的完整脚本代码
* 保存并启用

### 3️⃣ 使用
* 任意网页右上角（或你设置的位置）即可看到小窗
* 操作方式：
  * **拖拽标题栏**移动
  * **点击标题栏**折叠/展开
  * 点击 **NY/LOC 徽章**切换显示时区

---

## ⚠️ 合规与风险提示（重要）

> 本脚本仅用于展示交易时段信息与倒计时提示  
> **不提供、不包含、不涉及任何交易功能**

* 不支持下单、撮合、资金划转、充值、提现
* 不构成投资建议或收益承诺
* 不对任何证券或交易平台作推荐

### 关于准确性
市场交易时间可能因以下原因变化：
* 半日市（提前收盘）
* 临时休市/突发事件
* 交易所规则更新或临时调整

👉 使用者应以交易所/券商官方公告与实际行情为准  
👉 本脚本仅建议用于 **时间提醒、学习与技术研究用途**

---

## 🛠 技术说明（给维护者）
* 纯原生 JS + DOM + CSS（无依赖）
* 时区与夏令时处理依赖 `Intl.DateTimeFormat`（DST-aware）
* 状态/倒计时始终以 **America/New_York** 推算
* 位置、折叠状态、显示时区模式使用 GM 存储（无则 fallback 到 localStorage）
* 假日算法为 best-effort：常见 NYSE 假日；不含半日市与临时休市
